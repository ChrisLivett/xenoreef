# Lesser Europan Shark

- Family: Europan Shark
- Original guide page: https://web.archive.org/web/20031219032012id_/http://alienfishexchange.com:80/guide/lesser.html

## Notes

The first predator discovered on Europa. By comparison with some of the others, it's tame. It can still wreak havoc in a small tank, though.
